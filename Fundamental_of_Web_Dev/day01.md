## üåê TOPIC 1: Internet Fundamentals

### What is the Internet?
The Internet is a global system that connects millions of computers and devices so they can share information and communicate with each other.

It allows users to access websites, send emails, stream videos, and use online services.

---

### Why is it called a ‚ÄúNetwork of Networks‚Äù?
The Internet is not a single network.  
It is made by connecting many smaller networks such as:
- Home networks
- Office networks
- ISP networks
- Data center networks

All these networks are linked together, forming one large global network.

---

### Brief History of the Internet
- The Internet started as **ARPANET** in 1969.
- It was created to allow reliable communication between computers.
- The idea was to keep communication working even if some parts failed.

This concept helped shape today‚Äôs distributed and reliable Internet.

---

### ISP (Internet Service Provider)
An ISP is a company that provides Internet access to users.

**What an ISP does:**
- Connects users to the Internet
- Assigns an IP address
- Routes data between user devices and websites

**Examples:** Jio, Airtel, BSNL

---

### Types of Internet Connections
- **Broadband:** Wired, moderate speed, commonly used in homes
- **Fiber:** High-speed, stable connection using optical fiber
- **Mobile Data:** Wireless Internet via mobile networks (4G/5G)
- **Hotspot:** Sharing mobile Internet with other devices

---

### Internet vs Intranet vs Extranet

| Type | Meaning | Example |
|---|---|---|
| Internet | Public network | Google, YouTube |
| Intranet | Private internal network | Company HR portal |
| Extranet | Limited access private network | Client or vendor portal |

## üîå Ethernet

### What is Ethernet?
Ethernet is a wired networking technology used to connect devices within a local area network (LAN) using cables.

---

### Why Ethernet is Used
- Stable and reliable connection
- Faster than wireless connections
- Widely used in homes, offices, and organizations

---

### Ethernet vs Wi-Fi

| Ethernet | Wi-Fi |
|---|---|
| Wired connection | Wireless connection |
| More stable | Can fluctuate |
| Less interference | Affected by obstacles |

## üåê TOPIC 2: Website & Web Application

### What is a Website?
A website is a collection of web pages under one domain, mainly used to provide information to users.

**Examples:**
- Company website
- Blog
- Portfolio website

---

### What is a Web Application?
A web application is a website that allows users to interact, input data, and perform actions such as logging in or submitting forms.

**Examples:**
- Gmail
- Facebook
- Online banking systems

---

### Static Website

**Meaning:**  
A static website shows the same content to every user.

**Characteristics:**
- Fixed content
- No database
- No user login
- Simple and fast

**Examples:**
- Portfolio website
- Event information site

---

### Dynamic Website

**Meaning:**  
A dynamic website displays content that can change based on user interaction or data.

**Characteristics:**
- Uses databases
- Supports login and forms
- Interactive and personalized

**Examples:**
- Social media platforms
- E-commerce websites

---

### Website vs Web Application

| Website | Web Application |
|---|---|
| Mainly informational | Action-oriented |
| Limited interaction | High interaction |
| Usually no login | Login commonly required |
| Read-focused | Task-focused |

---

### SPA (Single Page Application)

**Meaning:**  
An application where the page does not fully reload when users navigate.

**Characteristics:**
- Smooth experience
- Faster navigation
- Content updates dynamically

**Example:** Gmail

---

### MPA (Multi Page Application)

**Meaning:**  
An application where each user action loads a new page from the server.

**Characteristics:**
- Traditional structure
- Full page reloads
- Good for content-heavy sites

**Example:** News websites

---

### Web App vs Mobile App

| Web Application | Mobile Application |
|---|---|
| Runs in a browser | Installed on device |
| No installation required | Requires installation |
| Works on any device with browser | Platform-specific |

---

## üìå Summary
- The Internet connects multiple networks globally.
- Ethernet enables stable wired communication.
- Websites focus on information, while web applications focus on interaction.
- Static sites show fixed content; dynamic sites change based on data.
- SPAs update content without reloading; MPAs load new pages.
- Web apps run in browsers; mobile apps are installed on devices.
